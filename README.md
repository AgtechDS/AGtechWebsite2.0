## ğŸš€ Quick Start

### Prerequisiti
- Node.js 18+
- npm o yarn
- Git

### Installazione

1. **Clona il repository**
   ```bash
   git clone https://github.com/AgtechDS/AGtechWebsite2.0.git
   cd AGtechWebsite2.0
   ```

2. **Installa le dipendenze**
   ```bash
   npm install --legacy-peer-deps
   ```

3. **Avvia il server di sviluppo**
   ```bash
   npm run dev
   ```

4. **Apri il browser**
   ```
   http://localhost:5173
   ```

### Script Disponibili

